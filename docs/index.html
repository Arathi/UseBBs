安装UseBBs

在安装UseBBs之前，你应该先了解一下你的服务器是否能够安装UseBBs。

最低要求

  • 支持PHP(或CGI)的HTTP服务器
  • PHP 4.3.0, 包括 session, PCRE 与 mysql 等扩展
  • MySQL 3.23

推荐安装环境

  • GNU/Linux操作系统
  • 最新版Apache HTTPD，包括 mod_rewrite 功能
  • 最新版PHP 5，包括 session, PCRE, mysql(i), zlib 与 curl 等扩展
  • 最新版MySQL

UseBBs支持较新的PHP 5.4以及MySQL 5.5。

UseBBs也能在别的Web服务器，例如Microsoft IIS, lighttpd 以及 ABYSS。但是不能保证在这些Web服务器的低版本下仍然能正常工作。

注意: 你必须选择其中一种方式来安装。如果你不知道该选哪一种方式，那就用自动安装器。

自动安装

使用 自动安装器 基本上就是最好最快的安装UseBBs的方式。

  1. 将 config.php-dist 重命名为 config.php；
  2. 将所有文件上传到web服务器的指定目录下；
  3. 可选： 使用chmod命令修改 config.php 文件的访问权限为可写，使得安装器和UseBB的ACP可以对其进行修改；
  4. 用你的浏览器打开 安装向导；
  5. 按照安装向导中的说明操作。

手动安装

当然，你也可以手动安装UseBBs。

  1. 先后执行 install/schemas/mysql.sql 以及 install/usebb.sql 两个数据库脚本 （推荐使用数据库管理工具，例如 phpMyAdmin）；
  2. 将 config.php-dist 重命名为 config.php；
  3. 编辑 config.php 文件，调整一下数据库相关信息1；
  4. 上传除了 install 目录以外的所有文件到web服务器指定目录下；
  5. 可选： 使用chmod命令修改 config.php 文件的访问权限为可写，使得UseBB的ACP可以对其进行修改；
  6. 注册第一个用户，它将自动被设置为管理员。

1如果你使用的PHP 5以上版本，并且mysqli扩展可用，可以将数据库类型设置为 'mysqli'。

安装后

安装完成后，你可以登陆管理员账号并开始配置论坛。ACP(管理员控制面板)的链接位于页面的底端。

如果您的论坛对外公开，建议阅读一下反垃圾指南，并做相应的设置。默认情况下，反垃圾的功能均未开启。

开发者注记

UseBB 1 及其beta版本0.x是作为一个业余爱好项目来开发的，很大程度上只是为了提升PHP和MySQL的开发经验。此项目最先开始于2004年。

尽管UseBB 1 已被证实非常适用于小型社区，而且仍然在维护中，但在实现上并非一个良好的现代开发技术应用案例。例如，UseBB 1 缺乏面向对象思想，在设计上也有若干缺陷，其他网页应用程序的开发中应当尽量避免这些缺陷。

要记住UseBB的定位：轻量级的和——最重要的——经典的论坛系统。
